# claude-plugins

Marketplace of Claude Code plugins by BigJiggity.

## Install the marketplace

```text
/plugin marketplace add BigJiggity/claude-plugins
```

## Plugins

| Plugin | Description |
|---|---|
| [**Cairn**](./cairn) | Marks the trail and remembers the path. Wires GSD planning (`/gsd:*`) to the beads tracker (`bd`) — create, claim, and close tracked work — optionally mirrors issues to GitHub/GitLab/Jira/Asana/Azure Boards, and optionally makes the context-mode knowledge base intent-aware (memory scoped to the active issue + phase). Activates only in repos that use both GSD and beads. |

Install a plugin:

```text
/plugin install cairn@claude-plugins
```
